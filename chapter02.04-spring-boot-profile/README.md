# Spring Boot Profile Example 

Spring Boot Profile 구현 예제

### SpringBoot Application 실행

```shell
java -jar customer-0.0.1-SNAPSHOT.jar
```

### Profile별 application.properties  

1. Default Profile에 대한 Properties : application.properties
1. prd Profile에 대한 Properties : application-prd.properties

### Profile별 Configuration을 위한 Configuration Class

1. Default Profile에 대한 Configuration Class : ConfigurationLocal
1. prd Profile에 대한 Configuration Class : ConfigurationPrd
